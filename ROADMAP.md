## ✅ フェーズ 5: 実戦・効率化機能 (Completed v2.4)

### 5.1 クイック・アクション
- [x] **一覧画面での tel: / mailto: リンク実装**
  - 電話番号をタップして直接発信（tel: リンク）。
  - メールアドレスをタップしてメーラー起動（mailto: リンク）。
  - ホバー時に表示される小さいアクションボタン（📞 / ✉️）。

- [x] **詳細画面での拡張**
  - メール・電話・URL フィールド横に「open」ボタン（tel: / mailto: / https:）。
  - フォーム編集中も即座に連絡可能。

### 5.2 エクスポート機能
- [x] **5.2.1 vCard (スマートフォン連絡先用)**
  - 個別名刺の「連絡先へ追加（VCF）」ボタン（詳細画面ヘッダー）。
  - FN (Full Name), ORG, TITLE, TEL, EMAIL, ADR, URL, NOTE を RFC 2426 に準拠して出力。
  - iOS / Android の標準連絡先アプリへ直接インポート可能（.vcf 形式）。

- [x] **5.2.2 CSV (PC / Excel / Google Sheets 用)**
  - 一覧画面ヘッダーに「CSV」ボタン。
  - 全名刺データを一括ダウンロード（フィルタ後の件数）。
  - 列: 氏名、かな、会社、部署、役職、メール、電話、郵便番号、住所、URL、交換日、メモ、位置情報。
  - RFC 4180 準拠の CSV エスケープ（カンマ・改行・引用符対応）。

## ⚠️ 実装上の鉄則
- **Auth Scope**: Gemini Key はログイン中のみ localStorage に展開、ログアウトで消去。
- **Density**: モバイル俯瞰性最優先（余白最小化）。
- **Privacy**: 元画像は解析後即廃棄。
